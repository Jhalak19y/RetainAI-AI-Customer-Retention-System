# 🚀 RetainAI

AI-Powered Customer Retention Intelligence Platform

## 📌 Overview

RetainAI predicts customer churn risk using Machine Learning and provides AI-powered retention insights and recommendations.

## ✨ Features

- Customer churn prediction
- AI risk scoring
- Interactive analytics dashboard
- Login & signup system
- Prediction history tracking
- Business insights & recommendations
- Beautiful Streamlit UI

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly
- Joblib

## 📊 Business Problem

Businesses lose customers because they fail to identify churn risks early. RetainAI helps companies detect high-risk customers and improve retention strategies.
